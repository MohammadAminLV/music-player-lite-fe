# Simple Music Plater app Front-end

## Quick start
1. Copy `.env.example` to `.env` and fill the values.
2. Install dependencies:
```bash
npm install
```
3. Run the app in dev mode:
```bash
npm run dev
```
4. For production build:
```bash
npm run build
```


## Docker
Build and run:


```bash
docker-compose up --build
```
